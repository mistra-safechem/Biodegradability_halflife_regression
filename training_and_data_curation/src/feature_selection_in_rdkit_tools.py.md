# About Feature Selection
Features calculated for all data sets were based on RDKit 2022 version, using the 
RDkit Descriptors and MACCSkeys, even though Rdkit 2025 is used in the environment. 
Thus the 2025 features were filtered based on the 2022 list (available as txt files in folder).

---
## Feature Selection
Following features were removed:

["ExactMolWt", "HeavyAtomMolWt", "FpDensityMorgan1", "FpDensityMorgan2", "FpDensityMorgan3"].

This, since ‘MolWt’ was deemded sufficient for use as weight descriptor and FpDensityMorgan 
deemed unnecessary since no Fingerprints were involved.

As additional descriptor, aqueous solubility (logS) was estimated using the ESOL model of 
Delaney (J. Chem. Inf. Comput. Sci. 2004, 44, 1000. https://doi.org/10.1021/ci034243x), 
based on molecular logP, molecular weight, number of rotatable bonds, and aromatic proportion. 
The original published equation is: 

logS = 0.16 - 0.63·clogP - 0.0062·MW + 0.066·RB - 0.74·AP
