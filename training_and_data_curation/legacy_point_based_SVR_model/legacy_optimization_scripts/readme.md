# Archiving legacy optimization scripts for point based model
This folder contains legacy optimization scripts for the point based model. 
These scripts were used during the development and optimization of the model, 
and they represent various stages of the modeling process. 
They may include different versions of the code, experiments with 
different parameters, and results from various runs using different models.

Kept here for reference and reproducibility.

## Data
These scripts use the same data as the top level curation pipeline.

## refactoring and adjustments
A best effort has been made to refactor the scripts to work from this folder 
without adjustments, after files had been moved around without reexecuting 
the scripts; this to preserve the original code and results as much as 
possible for archiving and reference purposes.
